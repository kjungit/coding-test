# 연산자(Operators)

연산자(operator)는 `+`, `-`, `*`, `=`, `===`, `!` 같은 기호로, 이는 더하기, 빼기, 곱하기, 할당, 비교, 부정 등의 의미를 가집니다.

## 산술(Arithmetic)

```js
console.log(1 + 2); // 3
console.log(5 - 7); // -2
console.log(3 * 4); // 12
console.log(10 / 2); // 5
console.log(7 % 5); // 2
```

## 할당(Assignment)

```js
let a = 3;

// a = a + 2
a += 2;

// a = a - 2
a -= 2;

// a = a * 2
a *= 2;

// a = a / 2
a /= 2;

// a = a % 2
a %= 2;

console.log(a); // 5, 1, 6, 1.5, 1
```
